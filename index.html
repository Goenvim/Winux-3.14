Mac

Launchers, window managers, and quiet enhancements for macOS.

  • Alfred
    Launcher Automation Paid

    Keyboard launcher with workflows, snippets, and clipboard search. Replace reaching for the mouse with two keys.

  • Raycast
    Launcher Extensions Free

    Command palette for your Mac. Deep integrations, extensions, and a modern UI.

  • Rectangle
    Windows Free

    Window snapping with simple keyboard shortcuts. Keeps multiple displays under control.

  • BetterTouchTool
    Input Automation Paid

    Customize trackpad, mouse, keyboard, and Touch Bar. Create gestures for everything.

  • iTerm2
    Terminal Free Open source

    Terminal emulator with panes, search, triggers, and profiles. A default for many macOS developers.

  • Homebrew
    Package manager Free Open source

    The package manager that quietly installs almost everything else.

  • CleanShot X
    Screenshots Paid

    Screen capture, annotation, and recording with a restrained interface.

  • Dropzone
    Files Paid

    Drag-and-drop shelf for moving and automating files across apps and services.

  • Magnet
    Windows Paid

    Alternative window manager with simple snapping presets, tuned for macOS.

  • Hazel
    Files Automation Paid

    Watches folders and automatically renames, files, and archives based on rules.

  • Hammerspoon
    Automation Experimental Open source

    Lua automation layer on top of macOS APIs. Scriptable control over windows, audio, and more.

  • Karabiner-Elements
    Keyboard Open source Experimental

    Deep keyboard remapping with layers and hyper keys. For people who love keymaps.

  • Amphetamine
    Utility Free

    Keeps your Mac awake on demand, with profiles and triggers.

Windows

Small utilities that make Windows feel intentional.

  • Microsoft PowerToys
    Power tools Free Open source

    Collection of utilities including FancyZones, PowerRename, and a quick launcher.

  • Everything
    Search Free

    File search that feels instant, no matter how large the drive.

  • ShareX
    Screenshots Recording Free Open source

    Screenshot and screen recording with powerful workflows and destinations.

  • AutoHotkey
    Automation Scripting Experimental Free

    Scripting language for hotkeys and automation. Old-school, but endlessly flexible.

  • EarTrumpet
    Audio Free

    Per-app volume control that should have shipped with Windows.

  • Wox
    Launcher Free

    Minimal launcher for Windows inspired by Alfred, with plugin support.

  • Flow Launcher
    Launcher Free Open source

    Extensible launcher for Windows with plugins, theming, and web search.

  • DisplayFusion
    Displays Paid

    Multi-monitor control, taskbars, and window rules for ultra-wide and multi-display setups.

  • Chocolatey
    Package manager Free

    Package manager for Windows that brings some of the convenience of Homebrew.

  • Listary
    Search Paid

    Inline search and quick switcher for files and folders inside Explorer and open/save dialogs.

  • tmux
    Terminal Open source

    Terminal multiplexer for panes, sessions, and remote workflows.

  • fzf
    Search Open source

    Fuzzy finder that turns the terminal into a quick navigator.

  • ripgrep
    Search Open source

    Fast code search through large repositories and logs.

  • btop
    Monitoring Open source

    Colorful, information-dense resource monitor that still feels calm.

  • Syncthing
    Sync Free Open source

    Private, peer-to-peer file sync across machines and operating systems.

  • i3 / Sway
    Window manager Experimental Open source

    Tiling window managers that turn the desktop into a keyboard-driven grid.

  • Alacritty
    Terminal Open source

    GPU-accelerated terminal emulator focused on simplicity and speed.

  • Zellij
    Terminal Experimental Open source

    A modern terminal workspace with tabs, panes, and session management.

  • Dunst
    Notifications Open source

    Lightweight notification daemon that stays out of the way.
